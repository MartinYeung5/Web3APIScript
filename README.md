# Web3APIScript

## Install nodejs:
```bash
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install nodejs
```

## Install ane setup test environment
```bash
npm install -g ethereumjs-testrpc
testrpc
```

## Execute script
```bash
nodejs scripts/<script.js>
```

## Web3 Version:
0.20.3
